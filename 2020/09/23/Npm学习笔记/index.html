前言

npm（全称 Node Package Manager，即“node包管理器”）是Node.js默认的、用JavaScript编写的软件包管理系统。（维基百科）

安装Npm

  • 安装Nodejs默认会预装Npm

模块位置

本地位置

  • 在终端当前目录下会产生node_modules目录

全局位置

  • MacOS会在/usr/local/lib/目录下产生node_modules目录

安装当前项目所依赖的所有模块

  • 安装所有package.json中所依赖的第三方模块
