  • 生成令牌或解析令牌时报错：token signature is invalid: key is of invalid type

原因一

  • 生成令牌或解析令牌时应当使用[]byte类型的key，而不是string类型或其他类型

解决问题

  • 生成令牌或解析令牌时应当使用[]byte类型的key

原因二

  • 加密方式应当是jwt.SigningMethodHS256而不是jwt.SigningMethodES256
