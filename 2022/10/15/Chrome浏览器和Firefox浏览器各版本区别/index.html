前言

Chrome浏览器和Firefox浏览器各版本区别

Chrome浏览器

  • 中文昵称：谷歌浏览器
版本名称 中文别名 适合人群 更新频率 区别
Chromium Chrome 社区版 极客用户 - Chrome浏览器开源版本，没有Google自家的服务，没有商业闭源插件（例如Flash）
Chrome Stable Chrome 正式版 普通用户 每月更新1次 Chrome浏览器正式版
Chrome Beta Chrome 尝鲜者版 尝鲜用户 每周更新1次 相比于正式版提前1个版本，最接近正式版的版本，是正式版下一次更新后的版本
Chrome Dev Chrome 开发者版 开发者 每周更新1次 相比于正式版提前1个版本，最接近正式版的版本，是正式版下一次更新后的版本
Chrome Canary Chrome 探索者版 极客用户 每天更新1次 相比于正式版提前2个版本，更新频率最高的版本，可以与其他版本并存
Chrome Enterprise Chrome 企业版 企业用户 - 提供企业级支持和工具

FireFox浏览器

  • 中文昵称：火狐浏览器
版本名称 中文别名 适合人群 更新频率 区别
Firefox Firefox 正式版 普通用户 每月更新1次 Firefox浏览器正式版
Firefox Beta Firefox 测试版 尝鲜用户 每周更新3次 最接近正式版的版本，是正式版下一次更新后的版本
Firefox Developer-edition Firefox 开发人员版 开发者 每周更新3次 最接近正式版的版本，是正式版下一次更新后的版本
Firefox Nightly Firefox 每日更新版 极客用户 每日更新1次 更新频率最高的版本
Firefox Extended Support Release Firefox 扩展支持发布版 企业用户 每42周更新1次 更新频率最低的版本
Firefox Focus Firefox 隐私浏览器 注重隐私用户 - 无浏览痕迹，但只有移动端版本

下载地址

  • https://www.mozilla.org/zh-CN/firefox/all/#product-desktop-release
