前言

波形图学习笔记

题型1

  • 给出D触发器及输入，画出波形图

示例：D触发器初始状态为0，已知CP和A的波形，画出输出Q的波形

  1. 在CP向下画虚线
  • 如果C1接口处没有圈，则在CP左侧画虚线
  • 如果C1接口处有圈，则在CP右侧画虚线


  1. 判断虚线与A的焦点是高位还是低位
  • 如果是高位，则取1
  • 如果是低位，则取0
  • 如果判断不出，则取向左偏移一点的值
A的值
1
1
0
1
  1. 依次将A的值带入到触发器的A处
  • 如果进入触发器的值1，则有效
  • 如果进入触发器的值0，则无效
A的值 进入触发器的值 进入触发器是否有效
1 0 无效
1 0 无效
0 1 有效
1 0 无效
  1. 判断Q在虚线右侧的值Qn+1
  • 如果进入触发器有效，则为1
  • 如果进入触发器无效，则为0
A的值 进入触发器的值 进入触发器是否有效 Q在虚线右侧的值Qn+1
1 0 无效 0
1 0 无效 0
0 1 有效 1
1 0 无效 0

由此得出特性方程：Qn+1=D

  1. 由得到的表画波形图

题型2

  • 给出JK触发器及输入，画出波形图

示例：JK触发器的初始状态为0，已知J、K的波形图，画出输出Q的波形图

  1. 在CP向下画虚线
  • 如果C1接口处没有圈，则在CP左侧画虚线
  • 如果C1接口处有圈，则在CP右侧画虚线


  1. 判断虚线与J、K的焦点是高位还是低位
  • 如果是高位，则取1
  • 如果是低位，则取0
  • 如果判断不出，则取向左偏移一点的值
J、K的值
J 1
K 0
J 0
K 1
J 0
K 0
  1. 依次将J、K的值带入到触发器的J、K处
  • 如果进入触发器的值1，则有效
  • 如果进入触发器的值0，则无效
J、K的值 进入触发器的值
J 1 有效
K 0 无效
J 0 无效
K 1 有效
J 0 无效
K 0 无效
  1. 由对照表得到结果表

由题可知虚线左侧的初始值为0，然后依次根据对照表，由左侧的值Qn得到右侧的值Qn+1

对照表
J值是否有效 K值是否有效 Q在虚线右侧的值Qn+1
有效 无效 1
无效 有效 0
无效 无效 不变
有效 有效 取反
结果表
J、K的值 进入触发器的值 Q在虚线左侧的值Qn Q在虚线右侧的值Qn+1
J 1 有效 0 1
K 0 无效
J 0 无效 1 0
K 1 有效
J 0 无效 0 0
K 0 无效

由此得出特性方程：Qn+1=J·Qn+K·Qn

  1. 由结果表画波形图

题型3

  • 给出多个相连的触发器，画出波形图

示例：每个触发器的初始状态都为0，已知CP的波形，画出在CP作用下的Q0和Q1的波形

  1. 在CP向下画虚线
  • 如果C1接口处没有圈，则在CP左侧画虚线
  • 如果C1接口处有圈，则在CP右侧画虚线

  1. 根据题目已知和连线的输入，得到前两个状态的值，并补波形图

  1. 将上一次的输出改为下一次的输入，得到两个状态的值，并补波形图

  1. 反复上一步的操作，直到补全波形图
