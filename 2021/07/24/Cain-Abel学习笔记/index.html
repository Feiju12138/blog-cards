前言

Cain & Abel is a password recovery tool for Microsoft Operating Systems.（官网镜像）

本文仅用于网络信息防御学习

安装

  • 开始安装->Next

  • Next

  • 设置安装路径->Next

  • 设置开始菜单快捷方式->Next

  • 开始安装->Next

  • 安装完成->Finish

安装WinPcap

  • 安装WinPcap

  • Next

  • 同意协议->I Agree

  • 设置开启自启->Install

  • 安装完成->Finish

网络嗅探

  • 首次点击Start/Stop Sniffer按钮，会弹出嗅探的网卡选择

  • 选择嗅探的网卡->确定

  • 再次点击Start/Stop Sniffer按钮，开始嗅探

  • Sniffer->Scan MAC Addresses

  • 勾选Rande->OK

ARP欺骗

  • APR->Add list->分别选择两台互相通信的主机进行欺骗->OK

  • Start/Stop APR

  • 此时攻击者作为中间人加入到了两台主机的通信中

中间人攻击

  • Passwords->查看截获的密码
