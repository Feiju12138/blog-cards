前言

在Windows Server 2003部署PPTP VPN服务
端口号TCP1123

准备工作

  • 一台有两块网卡的电脑，一块连接公网，一块连接外网

启用服务

  • 开始->所有程序->管理工具->路由和远程访问

  • 右键主机->配置并启用路由和远程访问

  • 下一步

  • 勾选自定义配置->下一步

  • 勾选VPN访问、拨号访问、NAT和基本防火墙->下一步

  • 完成

  • 是

创建用户并添加权限

  • 右键我的电脑->管理

  • 本地用户和组->用户->在空白处右键->新用户

  • 设置用户名和密码->创建

  • 关闭

  • 右键刚刚创建的用户->属性

  • 拨入->远程访问权限勾选允许访问->确定

配置下发IP地址的地址池

  • 右键主机->属性

  • IP->IP地址指派勾选静态地址池->添加

  • 配置地址范围->确定

  • 确定

开启NAT地址转换

  • 主机->IP路由选择->NAT/基本防火墙->空白处右键->新增接口

  • 选择内网的网卡->确定

  • 勾选在公用接口连接到Internet->勾选在此接口上启用NAT->确定

连接VPN（WindowsXP/WindowsServer2003）

创建一个连接

  • 在另一台公网电脑->右键网上邻居->属性

  • 双击新建连接向导

  • 下一步

  • 勾选连接到我的工作场所的网络->下一步

  • 勾选虚拟专用网络连接->下一步

  • 指定一个公司名->下一步

  • 指定VPN服务器的主机名或IP地址

  • 下一步

  • 完成

通过拨号连接VPN

  • 输入用户名和密码->连接

访问内网

完成
