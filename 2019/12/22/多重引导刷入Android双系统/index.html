【笔记】多重引导刷入 Android 双系统

发表于2019-12-22|更新于2024-08-06
|阅读量:

前言

通过多重引导（Dual Boot Patcher）刷入Android双系统

准备工作

  • 两种ROM刷机包

我使用的分别是

魔趣——主系统（传送门）
MIUI——副系统（传送门）

  • Dual Boot Patcher安装包

刷入主系统

  • 首先通过adb刷入主系统

传送门

为副系统ROM包打补丁

安装 Dual Boot Patcher

获取 Root 权限

  • 在菜单选择ROMs

  • 获取成功后，显示当前只有一个主系统

为副系统ROM包打补丁

  • 在菜单选择Patch Zip File

  • 点击加号选择需要打补丁的系统ROM包

  • 设置参数->continue

Device：klet（选择手机型号）
Partition configuration：Data slot（刷入Data分区）->miui（设置名称）

  • 选择打补丁后的ROM存储位置及名称

为了区分，我在原名称后添了2

  • 开始打补丁

刷入副系统

  • 进入TWRPRecovery模式

  • 选择高级->选择ADB线刷

  • 重启后自动进入新系统

系统切换

  • 在新系统同样安装Dual Boot Patcher

  • 在菜单选择ROMs

  • 选择下次开机要启动的系统

完成
