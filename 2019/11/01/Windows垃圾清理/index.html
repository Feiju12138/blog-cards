前言

Windows垃圾清理脚本

通过垃圾清理脚本清理垃圾

创建文件

  • 在任意文件夹中空白处，右键->新建->文本文档

修改文件扩展名

  • 右键->重命名->”垃圾清理.bat”->(是否更改扩展名)是

写入代码

  • 右键->编辑->写入代码->保存->关闭

运行程序

  • 右键->以管理员方式运行

完成

手动删除无用文件

在系统设置中删除临时文件

  • 设置->系统->存储->临时文件

  • 勾选需要删除的临时文件->删除文件
